# Page View Time Series Visualizer ._.7

### [FreeCodeCamp - Data Analysis with Python Project]

This project visualizes time series data from a dataset containing the number of daily page views on the freeCodeCamp forum from 2016-05-09 to 2019-12-03.

---

## 🚀 Deployment Achievements
* **Data Sanitization:** Successfully filtered top and bottom 2.5% to eliminate statistical noise.
* **Advanced Visualization:** Implemented line plots, monthly average bar charts, and box plots.
* **Tech Stack:** Developed using the elite trio: **Pandas**, **Matplotlib**, and **Seaborn**.
* **Status:** 11/11 Tests passed (Verified by `test_module.py`).

---

## 📊 Generated Visualizations

### 1. Line Plot
Visualizes the daily growth of views, allowing for long-term trend identification.

### 2. Bar Plot
Displays monthly average views grouped by year, facilitating seasonal comparisons.

### 3. Box Plots
Two diagrams analyzing distribution: one by year (Trend) and another by month (Seasonality).
